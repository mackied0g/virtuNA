# virtuNA

virtu-NA, or Virtual NA, is an open source web app to increase the accessibility and retention of Narcotics Anonymous. 

Currently, we are using:

- Node.js (10.15.2)
- React.js (16.9.0)
- WebSockets (1.0.30)
